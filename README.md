# ✂️ Hairdresser — Responsive Landing Page

A responsive landing page for a hairdresser salon. Built with pure HTML, CSS, and JavaScript — no frameworks.

---

## ✨ Features

- 📱 Fully responsive & mobile-friendly layout
- 🎨 Clean and modern design
- ⚡ Pure HTML/CSS/JS — no dependencies
- 🖼️ Image gallery section
- 📐 Adaptive grid layout for all screen sizes

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Page structure |
| CSS3 | Styling, Flexbox/Grid, media queries |
| JavaScript | UI interactions |

---

## 🚀 Getting Started

No build tools needed — just open the file in your browser:

```bash
# Clone the repository
git clone https://github.com/yarick93/hairdresser.git

# Open in browser
open index.html
```

Or simply download the ZIP and open `index.html` directly.
